CSS

CSS (Cascading Style Sheets) é a Linguagem usada para definir a aparência e o layout das páginas feitas com HTML, controlando cores, fontes, espaçamentos, tamanhos e posicionamento dos elementos; em resumo, enquamto o HTML estrutura o conteúdo, o CSS deixa tudo visualmente organizado e bonito.
